Ce script permet d'interroger et de moissonner l'entrepôt Cocoon pour récupérer toutes les métadonnées de la collection Pangloss.
Ca fournit un fichier xml contenant les métadonnées de chaque ressource de la collection (une sorte de catalogue)

Pour lancer le script : 
sh pangloss.sh
ou
pangloss.bat

********


This script is used to harvest Cocoon and retrieve Pangloss Collection's metadata.
It will provide an xml file containing the metadata of the Pangloss Collection's resources (a kind of catalogue)

Command:

sh pangloss.sh
or
pangloss.bat
